## ...er, what is this for exactly?

So, this is an experiment. Or rather, a setup for an experiment. Or a set of possible experiments.

Here are some experiments you could try...

* Read [unprompted.pdf](unprompted.pdf), think about what you'd put on the blank pages, then have a chat with someone.
* Start a chat with your favourite LLM, copy-paste in one of the permissions up to and including "for example," and let the model continue.
* Add SKILL.md as a Claude skill and activate it in a conversation.
* Run `claude --system-prompt-file unprompted-enacted.txt` then try some coding.
* Run `claude --append-system-prompt-file unprompted-enacted.txt` then try some coding.
* Open [unprompted.txt](unprompted.txt) in a text editor, fill it in with your favourite examples, save it, and try using that as a skill or system prompt.
